# 广告行业纵向趋势分析 SKILL

> 角色：腾讯广告战略分析师
> 方法论对标：McKinsey Revenue Bridge + BCG TSR Decomposition + Minto Pyramid Principle
> 输入：同一公司多期的单期财报分析产出（8 章详报 markdown）
> 输出：纵向趋势分析报告（markdown）+ 趋势摘要卡片（JSON）

---

## 一、前置条件

### 1.1 输入要求
- **最少 2 期**同一公司的单期财报分析产出（跨季度或跨年度均可）
- 推荐 4-8 期以获得完整趋势视角（如连续 4 个季度 或 连续 2-3 个财年）
- 每期产出须包含：revenue、net_income、eps、gross_margin、operating_cf 等核心指标

### 1.2 广告行业专用指标体系
除通用财务指标外，需额外提取或推算以下广告业务指标：

| 指标分类 | 具体指标 | 定义 | 数据来源优先级 |
|---------|---------|------|--------------|
| 用户规模 | DAU/MAU | 日活/月活用户数 | 财报原文 > 分析师电话会 > 行业估算 |
| 用户规模 | DAU/MAU 增速 | YoY 和 QoQ 变化率 | 计算得出 |
| 变现效率 | ARPU | 单用户平均收入 = 广告收入 ÷ DAU | 计算得出 |
| 变现效率 | eCPM | 每千次展示有效收入 | 财报原文 > 行业报告推算 |
| 广告负载 | Ad Load Rate | 单位内容中的广告密度 | 财报电话会 > 行业估算 |
| 广告负载 | Impressions | 总广告展示量 = DAU × 人均 session × 每 session 广告数 | 推算 |
| 收入结构 | 广告收入占比 | 广告收入 ÷ 总收入 | 计算得出 |
| 收入结构 | 广告产品分布 | 搜索广告/信息流/短视频/效果广告各自占比 | 财报分项 > 分析师估算 |

### 1.3 数据可得性分级
不同公司的数据披露粒度差异大，需标注数据置信度：
- **A 级（高置信）**：直接来自财报原文或 SEC/港交所/东交所公告，精确到小数点后 2 位
- **B 级（中置信）**：来自管理层电话会口头披露、分析师日 presentation，可能有四舍五入
- **C 级（低置信）**：来自行业报告推算、第三方估算、媒体报道，须明确标注"估算"
- **N/A**：数据不可得（如字节跳动未上市，部分指标无公开数据）

## 二、纵向分析方法论（5 步流程）

### Step 1：时间序列数据矩阵构建

将多期数据整理为标准化矩阵：

```
| 指标 \ 时期 | FY23Q1 | FY23Q2 | FY23Q3 | FY23Q4 | FY24Q1 | FY24Q2 | ... |
|-------------|--------|--------|--------|--------|--------|--------|-----|
| 总收入 ($B) | xx.xx  | xx.xx  | xx.xx  | xx.xx  | xx.xx  | xx.xx  |     |
| 广告收入 ($B)| xx.xx | xx.xx  | xx.xx  | xx.xx  | xx.xx  | xx.xx  |     |
| DAU (M)     | x,xxx  | x,xxx  | x,xxx  | x,xxx  | x,xxx  | x,xxx  |     |
| ARPU ($)    | xx.xx  | xx.xx  | xx.xx  | xx.xx  | xx.xx  | xx.xx  |     |
| 毛利率 (%)  | xx.xx  | xx.xx  | xx.xx  | xx.xx  | xx.xx  | xx.xx  |     |
| ...         |        |        |        |        |        |        |     |
```

**要求**：
- 所有货币统一为同一币种（默认 USD，中国公司用 RMB 并标注汇率）
- 所有百分比保留 2 位小数
- 缺失数据标注 N/A 并注明原因
- 每个数据点标注置信度等级（A/B/C）

### Step 2：收入桥分解（McKinsey Revenue Bridge）

**核心方法：Price-Volume-Mix 三因素分解**

对于广告行业，将经典 PVM 模型适配为：

```
广告收入变化 = 用户量效应 + 变现效率效应 + 结构效应
```

具体公式：

**用户量效应（Volume Effect）**
= (本期 DAU - 上期 DAU) × 上期 ARPU
> 回答：增长是因为用户盘子变大了吗？

**变现效率效应（Price Effect）**
= (本期 ARPU - 上期 ARPU) × 本期 DAU
> 回答：增长是因为单用户价值提升了吗？

**结构效应（Mix Effect）**
= 总收入变化 - 用户量效应 - 变现效率效应
> 回答：增长是因为高价值广告产品（如短视频、效果广告）占比提升了吗？

**进一步拆解 ARPU 变化**：
```
ARPU 变化 = eCPM 变化贡献 + Ad Load 变化贡献
其中：
  eCPM 变化贡献 = (本期 eCPM - 上期 eCPM) × 本期人均 impression
  Ad Load 变化贡献 = (本期人均 impression - 上期人均 impression) × 上期 eCPM
```

**瀑布图（Waterfall）表达顺序**：
```
上期广告收入 → (+/-)用户量效应 → (+/-)eCPM效应 → (+/-)Ad Load效应 → (+/-)结构效应 → 本期广告收入
```

### Step 3：利润桥分解

```
营业利润变化 = 收入变化贡献 + 毛利率变化贡献 + 运营杠杆贡献
```

其中：
- **收入变化贡献** = (本期收入 - 上期收入) × 上期营业利润率
- **毛利率变化贡献** = (本期毛利率 - 上期毛利率) × 本期收入
- **运营杠杆贡献** = 总营业利润变化 - 收入变化贡献 - 毛利率变化贡献

重点关注广告业务的成本结构变化：
- 流量获取成本（TAC）占广告收入比例变化
- 内容成本 / 创作者分成变化
- AI/算法基础设施投入变化
- 销售与营销费用率变化

### Step 4：趋势判断与结构性拐点识别

**4.1 趋势分类**

对每个核心指标，判断其处于以下哪个阶段：
- **加速增长**：YoY 增速连续 2 期以上提升
- **稳态增长**：YoY 增速波动在 ±3pp 以内
- **减速增长**：YoY 增速连续 2 期以上下降，但仍为正
- **拐点/转负**：从正增长转为负增长
- **底部反弹**：从负增长转为正增长
- **持续收缩**：连续 2 期以上负增长

**4.2 季节性调整**

广告行业有明显季节性（Q4 节日旺季、Q1 春节/新年淡季），趋势判断须：
- 优先用 YoY（同比）而非 QoQ（环比）
- 如用 QoQ，必须同时给出去年同期的 QoQ 作对照
- 标注异常事件影响（如 COVID、政策变化、IDFA/隐私政策）

**4.3 结构性变化信号**

以下信号提示可能存在结构性变化（非周期性波动）：
- 广告收入增速与 DAU 增速出现 ≥5pp 持续背离（变现加速/减速）
- 某一广告产品占比连续 3 期单向变化 ≥2pp（产品结构迁移）
- 毛利率连续 3 期单向变化 ≥1pp（成本结构变化）
- ARPU 增速显著超过或落后于行业中位数（竞争力信号）

### Step 5：BCG 价值创造分解（适用于上市公司）

对上市公司，补充 BCG TSR 四因素分解：

```
TSR = 收入增长贡献 + 利润率变化贡献 + 估值倍数变化贡献 + 现金流贡献
```

- **收入增长贡献**：广告收入 CAGR × 权重
- **利润率变化贡献**：EBITDA margin 或 Operating margin 变化
- **估值倍数变化贡献**：EV/Revenue 或 P/E 变化反映市场预期
- **现金流贡献**：FCF yield、股息、回购

> 注：字节跳动未上市，此步骤标注 N/A

## 四、Critic 自检规则（6 条硬 gate）

| # | 规则 | 检查方式 |
|---|------|---------|
| 1 | 引用闭环 | 每个数字必须可溯源到 Appendix 脚注或数据矩阵 |
| 2 | 数字精度 | 所有百分比 ≥2 位小数，货币 ≥2 位小数 |
| 3 | 空洞词禁令 | "大幅/显著/明显"后 4 字内必须出现具体数字 |
| 4 | 收入桥平衡 | 用户量效应 + 变现效率效应 + 结构效应 = 总收入变化（允许 ±0.01 舍入误差）|
| 5 | 置信度标注 | 每个关键数据点必须标注 A/B/C/N/A 等级 |
| 6 | 腾讯视角 | So-What 部分必须包含对腾讯广告的具体启示，不能只是泛泛而谈 |

最多重写 2 轮。如 2 轮后仍未通过，标注未通过项并交付，由人工 review。

## 五、特殊处理

### 5.1 字节跳动/TikTok
- 未上市，无公开财报，数据主要来源：行业报告（eMarketer、Sensor Tower）、媒体报道、字节官方零星披露
- 所有数据默认置信度为 C 级
- 跳过 BCG TSR 分解
- 在分析中明确标注"以下数据基于第三方估算，仅供趋势方向参考，不宜用于精确对比"

### 5.2 中国公司（阿里、快手、腾讯）
- 货币单位：RMB，同时标注按期末汇率折算的 USD 值
- 财年与自然年对齐关系须明确标注（阿里财年 3 月结束）
- 中国广告市场季节性模式与全球不同（双 11、春节、618 等）

### 5.3 数据缺失处理
- 如某期缺失某指标，在数据矩阵中标注 N/A
- 收入桥分解中，缺失的因素归入"结构效应"并注明
- 不做外推或插值填补缺失数据
